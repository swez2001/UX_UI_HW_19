My UX Based Skills

User Research Processes

For user research I explored different websites, colour themes & topics for inspiration and I’ve used tools such as Figma, Miro and Invision to create designs & clipboards & Mood Boards to later use for the final prototypes. This process also includes empathising & interviewing target users to find out what functions are required for a product that will be prototyped.

User Interface Design & Prototyping

I predominately used Figma for constructing all the prototypes for all my case studies. The use of desktop & mobile size frames were used according to what type of prototype was being designed.

Web/Front-End Development

For Web Development or also known as Front-End Development, I have acquired skills for HTML, CSS and Javascript which have all been used to create this portfolio. This skillset is the most challenging of all the UX Skill set that I have faced.
